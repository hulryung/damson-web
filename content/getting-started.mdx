# Getting Started

This is the five-minute tour. Once Damson is [installed](/installation), open it
and try the workflows below ‚Äî they cover the controls you'll reach for every day.

## Split your first workspace

1. Press `‚åòD` to split the current pane into two side by side.
2. Press `‚åò‚áßD` to split one of them top-and-bottom.
3. Move focus with `‚åò‚å•` + an arrow key; rearrange panes with `‚åò‚áß` + an arrow key.

Panes form a tree, so you can nest splits as deeply as you like and Damson keeps
the geometry sensible. When you've found a layout you like, you don't have to
rebuild it by hand next time ‚Äî see below.

## Apply a layout in one keystroke

Instead of dragging splits around, press `‚åò‚åÉ1` to drop into the headline
**Columns 20 / 60 / 20** layout instantly. There are seven presets on
`‚åò‚åÉ1`‚Äì`‚åò‚åÉ7` ‚Äî read [Preset Layouts](/preset-layouts) for the full set.

## Navigate output like a document

Damson understands your shell's prompt marks (OSC 133), so you can move through
a session by command rather than by line:

- `‚åò‚Üë` / `‚åò‚Üì` jump between prompts.
- `‚åò‚áßC` copies the output of the last command.
- `‚åòF` searches the scrollback with a live overlay.
- **‚åò-click** any file path in the output to open it (with optional `:line`
  jump). See [Shell Integration](/shell-integration).

## Make it yours

Open **Settings** to set your font, theme, cursor, background opacity and blur,
ligatures, animations, scrollback size, and more. The full list is in
[Settings](/settings).
